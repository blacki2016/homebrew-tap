# Clawbar Homebrew Tap

Install Clawbar via Homebrew:

```bash
brew install blacki2016/tap/clawbar
```

Or use the Cask for the full GUI app:

```bash
brew install --cask blacki2016/tap/clawbar
```

## What is Clawbar?

Clawbar is a macOS menu bar app for monitoring AI token usage across 20+ providers including theclawbay, Claude, Codex, Gemini, Cursor, and more.

- **CLI**: `clawbar usage`, `clawbar cost`, `clawbar models`
- **GUI**: Menu bar widget showing live token usage
- **Auto-updates**: Via Sparkle

## Requirements

- macOS 14+ (Sonoma)
- Apple Silicon (ARM64)

## Uninstall

```bash
brew uninstall clawbar
brew uninstall --cask clawbar
```
